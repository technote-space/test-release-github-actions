## Installation
1. Setup workflow  
   e.g. `.github/workflows/pull_request.yml`
   ```yaml
   on: pull_request
   name: Pull Request
   jobs:
     assignAuthor:
       name: GitHub Actions Template
       runs-on: ubuntu-latest
       steps:
         - name: GitHub Actions Template
           if: github.event.action == 'opened'
           uses: technote-space/gh-actions-template@v1
           with:
             GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
   ```
